# DSA Problems 

This repository contains **Data Structures and Algorithms (DSA) problems** solved using **clean, efficient, and interview-oriented approaches**.  
It is intended for **coding interview preparation**, **problem-solving practice**, and **concept revision**.

---

##  What this Repository Contains

-  Common **DSA interview questions**
-  Optimized solutions with good **time & space complexity**
-  Problems from **arrays, strings, linked lists, trees, graphs, DP**
- Code written in a **structured and readable manner**

---

##  Topics Covered

- Arrays
- Strings
- Recursion
- Searching & Sorting
- Stack & Queue
- Linked List
- Trees & Binary Trees
- Graphs (BFS / DFS)
- Dynamic Programming
- Greedy Algorithms
- Bit Manipulation
